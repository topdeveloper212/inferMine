Computes a dataflow graph

Activate with `--lineage`.

Supported languages:
- C/C++/ObjC: No
- C#/.Net: No
- Erlang: Yes
- Hack: No
- Java: No
- Python: No



## List of Issue Types

The following issue types are reported by this checker:
- [LINEAGE_FLOW](/docs/next/all-issue-types#lineage_flow)
